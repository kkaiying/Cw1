# ILP CW3 - Drone Delivery Operations Dashboard

## Running with Docker
1. Load the Docker image named ilp_submission_image.tar
2. Run the container on port 8080 and wait until the application has started
3. Open your browser and type in http://localhost:8080 in the search bar (use Google Chrome as the browser)

## Running the source code directly
1. Navigate to this project's directory in your terminal
2. Run the project using mvn spring-boot:run and wait until the application has started
3. Open your browser and type in http://localhost:8080 in the search bar (use Google Chrome as the browser)

## Using the UI
- Click on any service point marker to view its drone and dispatch details
- Click on a drone to expand it and see its details
- Click on a dispatch ID to navigate to that dispatch point
- Click on a dispatch ID to view that dispatch's details
